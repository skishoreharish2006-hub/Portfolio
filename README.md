An ultra-modern, visually stunning, and highly interactive personal portfolio website designed to showcase developer skills with a futuristic aesthetic.

## ✨ Key Features

- **🌐 Interactive 3D Canvas**: Neural brain visualization and skill nodes built with Three.js and React Three Fiber.
- **🌌 Futuristic Design**: Dark theme with neon glows, glassmorphism, and smooth gradients.
- **⚡ Dynamic Animations**: Scroll-triggered transitions and micro-interactions powered by Framer Motion and GSAP.
- **🤖 AI Assistant UI**: Integrated AI assistant interface for an immersive user experience.
- **📱 Responsive Layout**: Fully optimized for all screen sizes using Tailwind CSS.
- **📊 Interactive Skill Nodes**: Visualized skill data using Recharts and 3D positioning.

## 🛠️ Tech Stack

- **Frontend**: [React](https://reactjs.org/)
- **3D Graphics**: [Three.js](https://threejs.org/), [React Three Fiber](https://docs.pmnd.rs/react-three-fiber/), [Drei](https://github.com/pmndrs/drei)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/), [GSAP](https://greensock.com/gsap/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Building Tool**: [Vite](https://vitejs.dev/)

## 🚀 Getting Started

### Prerequisites

- Node.js (Latest LTS version)
- npm or yarn

### Installation

1. **Clone the repository**:
   ```bash
   git clone <repository-url>
   cd My_PortFolio
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run the development server**:
   ```bash
   npm run dev
   ```

4. **Build for production**:
   ```bash
   npm run build
   ```

## 📂 Project Structure

- `/src/components/canvas`: 3D elements and Three.js scenes.
- `/src/components/sections`: UI sections like Hero, About, Projects, etc.
- `/src/components/ui`: Common UI components like Navbar, Footer, and AI Assistant.
- `/src/constants`: Configuration and static data for the portfolio.
- `/src/hooks`: Custom React hooks for animations and state management.
